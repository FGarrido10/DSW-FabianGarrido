        Proposito

        El desarrollo de esta herramienta tiene como objetivo conocer las etiquetas basicas de HTML asi como sus Atributos y respectivos valores.

        Etiquetas Basicas de HTML

        HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero.En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto.

        Las etiquetas basicas son: body: para el contenido.
        head: informacion sobre el documento.
        div: division dentro del contenido.
        a: para enlaces.
        strong: para poner el texto en negrita.
        br: para saltos de linea.
        h1 a h6:para titulos dentro del contenido.
        img: para añadir imagenes al documento.
        ol: para listas ordenadas.
        ul: para listas desordenadas.
        li: para elementos dentro de la lista.
        p: para paragrafos.
        span: para estilos dentro de una parte del texto.

        Atributos y Valores

        Un atributo en HTML es toda informacion que se adiciona siempre en la etiqueta de apertura con el objetivo de sumar mas datos o significado al elemento HTML que se esta trabajando.
